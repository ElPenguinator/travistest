## Backlog 
### Rôles
* **Utilisateur** : Personne utilisant l'application.
* **Développeur** : Membre d'un projet pouvant effectuer des modifications sur ce projet.
* **Product Owner** : Membre d'un projet pouvant simplement modifier les priorités.
